# 探索Adobe Target Floodgate {#get-started}


## 什麼是Floodgate？

Floodgate是功能管理和實驗系統，可協助應用程式以安全、漸進且較少風險的方式提供新體驗。

Floodgate不會一次為所有人開啟新功能，而是可讓產品團隊：

* 逐步發行功能
從一小部分使用者開始，然後隨著時間增加。 如果某些功能無法如預期運作，則可快速關閉功能，不需停機或重新安裝軟體。

* 將功能鎖定在特定對象
為選取的群組啟用新功能（例如Beta版計畫、特定區域或特定計畫），以便在變更可廣泛使用之前對其進行驗證和細化。

* 執行實驗和A/B測試
嘗試不同版本的功能（例如替代版面、流程或建議），並使用資料來衡量哪個功能效果更好，以指導產品決策。

* 降低重要版本的風險
大型變更可以分解為較小且受控制的轉出，因此問題更容易偵測、理解和修正，以免影響每個人。

* 快速回應問題
如果出現錯誤或效能問題，專案團隊可以只關閉受影響的功能，讓應用程式的其餘部分保持穩定。

在實務中，Floodgate可協助使用者更頻繁地使用功能改善，並減少中斷情形：新功能可更早送達、更仔細地測試，以及根據真實世界的意見回饋快速調整。
