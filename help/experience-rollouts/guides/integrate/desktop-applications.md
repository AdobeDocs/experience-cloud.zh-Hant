# 案頭應用程式 {#desktop-applications}

案頭應用程式可透過SDK與體驗轉出整合。 如需案頭棧疊專屬的整合指引，請聯絡您的Adobe代表。

>[!NOTE]
>
>案頭SDK檔案目前正在準備中，即將推出。

## 應用程式ID {#app-id}

案頭使用者端可以使用&#x200B;**產品代碼和產品版本**&#x200B;作為應用程式識別碼。
