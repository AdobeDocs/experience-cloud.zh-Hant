# 功能、功能群組和發行版本 {#features-feature-groups-releases}

體驗轉出提供四個用於管理功能轉出的成品。 選擇合適的受眾取決於轉出的範圍、涉及的團隊數量以及您需要的受眾目標定位功能。

## 四個成品 {#artifacts}

**功能標幟**
最原子化的單位。 控制單一應用程式的單一功能，該功能屬於一個團隊。 可以為定義的對象啟用或停用。

**功能群組**
屬於相同團隊的功能旗標集合。 允許在一個群組中跨多個應用程式管理多個旗標，作為單一單位。

**跨團隊功能群組**
將功能群組功能擴充至多個團隊和應用程式。 自助服務和支援豐富的對象條件，但不支援快取。

**版本**
專為跨多個團隊和應用程式的大型協調轉出所設計。 使用驗證權杖型對象目標定位。 支援伺服器SDK上的快取。

## 比較 {#comparison}

| | 功能標幟 | 功能群組 | 跨團隊功能群組 | 版本 |
|---|---|---|---|---|
| **管理對象所需的角色** | 產品發行擁有者 | 產品發行擁有者 | 功能管理員 | 發行管理員 |
| **應用程式** | 單一 | 多個（相同團隊） | 多重（跨團隊） | 多重（跨團隊） |
| **團隊** | 單一 | 單一 | 跨團隊 | 跨團隊 |
| **豐富對象條件** | ✓ | ✓ | ✓ | 有限 |
| **轉出百分比** | 與任何對象條件結合 | 與任何對象條件結合 | 與任何對象條件結合 | 與固定對象條件結合 |
| **A/B 測試** | ✓ | ✓ | ✗ | ✗ |
| **在伺服器SDK中快取** | 僅限預設開啟/關閉旗標 | 無快取 | 無快取 | 所有發行版本都快取於本機 |
| **自助服務** | ✓ | ✓ | ✓ | 需要支援要求 |
| **稽核軌跡** | ✓ | ✓ | ✓ | ✓ |

## 何時使用每個 {#when-to-use}

| 藍本 | 使用 |
|---|---|
| 針對一個應用程式測試或推出單一功能 | **功能標幟** |
| 在同一個團隊中協調多個功能，同時上線 | **功能群組** |
| 透過豐富的定位功能，協調不同團隊中不同應用程式的功能 | **跨團隊功能群組** |
| 跨多個團隊的大型協調發行，具有SDK層級的快取 | **版本** |
