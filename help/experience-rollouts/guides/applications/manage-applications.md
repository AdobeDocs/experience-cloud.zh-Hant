# 管理應用程式 {#manage-applications}

Experience轉出中的&#x200B;**應用程式**&#x200B;代表您要使用功能旗標控制的服務或產品。 您必須先將至少一個應用程式上線至主控台，才能建立功能標幟。

## 誰可以管理應用程式 {#permissions}

只有具有&#x200B;**管理員**&#x200B;角色的成員才能新增或修改應用程式。 如果您沒有此角色，請聯絡管理員。

## 新增應用程式 {#add-application}

若要新增應用程式，請參閱[將您的應用程式上線](onboard-your-application.md)，以取得逐步指示。
